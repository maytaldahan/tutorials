Overview

One platform, three tracks into modern AI on HPC

Tapis is an NSF-funded API platform for managing high-performance computing systems, submitting jobs, and moving data across HPC centers. It gives researchers a consistent way to manage compute resources, run jobs, and manage files — through APIs or through TapisUI, a graphical interface built on top of them.

This tutorial walks through Tapis from the ground up: what it is and why it exists alongside the national cyberinfrastructure ecosystem, how to run AI inference at scale with FlexServ, and how to apply computer vision models to real scientific data using the ICICLE Data Labeler.

FlexServ is a Tapis service that deploys and serves AI/ML models on HPC or VM resources, providing inference endpoints and a graphical interface for managing the service — the backbone of the afternoon's hands-on inference session on Vista.

The compute resources used in this tutorial are provided by funding from the National Science Foundation.

FAQ

Tapis, frequently asked

What is Tapis?+
Tapis is an NSF-funded API platform for managing high-performance computing systems, running jobs, and managing data across HPC centers. It lets users manage compute resources, submit jobs, and manage files through APIs or through TapisUI.
What is FlexServ?+
FlexServ is a Tapis service that allows you to deploy and serve AI/ML models on HPC or VM resources. It provides inference endpoints along with a graphical interface for managing the service.
Where can I ask questions after the tutorial?+
